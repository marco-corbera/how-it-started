Chapter 02 — 1989 to 1995

The First Web

In 1993, two organizations each held a technology that could become the internet's primary document system. One charged licensing fees. One gave it away for free. Within eighteen months, one of them had won. The other is a footnote nobody remembers.

The Decision That Changed Everything

April 30, 1993. CERN — the European physics lab in Geneva — releases a document with a quiet title: "WorldWideWeb: Summary." It announces that the web is now in the public domain. No patents. No licenses. No fees. Anyone, anywhere, can build a web server, write a browser, publish a page. Free. Forever.

At that exact moment, the competing system — Gopher, a clean and popular menu-based document network — was being quietly strangled. The University of Minnesota, which owned it, had just announced they would start charging licensing fees. Nothing outrageous. Just... fees.

It was enough. Developers abandoned Gopher almost overnight. By 1994 the web had ten times the traffic. By 1996, Gopher was dead.

One licensing decision. Two organizations. Two opposite choices. The entire shape of the modern internet followed from that gap.

Forgotten rival

Gopher was actually better than the web in 1992. Faster, more organized, easier to navigate. It had menus instead of links. More people used it. It lost not because the web was superior — but because the web was free.

The Man Who Started It

Four years earlier, a 33-year-old British physicist at CERN named Tim Berners-Lee had a problem. Not a physics problem — an organizational one.

CERN employed thousands of scientists from dozens of countries, rotating in and out on different projects. Every time someone left, their knowledge left with them. Documents existed but lived on incompatible systems. There was no way to link one piece of information to another.

Berners-Lee's insight was deceptively simple: the internet already connected these computers. What was missing was a layer on top — a system that let documents link to each other across different machines. Not a database. Not a file system. Something more like a web.

He wrote a proposal on March 12, 1989. His boss, Mike Sendall, read it and wrote four words on the cover before handing it back.

"Vague but exciting."
— Mike Sendall, CERN, 1989

That was the approval. No committee. No budget. No roadmap. Berners-Lee spent the next eighteen months building it alone, on a NeXT computer in a corner of CERN, largely in his spare time.

Three Problems. Three Inventions.

To make the web work, Berners-Lee had to solve three distinct problems simultaneously. The solutions he came up with are still the foundation of every website you visit today.

Invention 1

HTML — HyperText Markup Language

A language for describing documents. Not programming — no logic, no decisions, no loops. Just structure. Tags tell the browser what content is: a heading, a paragraph, an image, a link.

The killer feature was a single tag: <a href="...">. A hyperlink. Click it and the browser fetches a different document — from anywhere on the network. That mechanic, click-and-follow, is what made the web a web and not just a file system.

Invention 2

URL — Uniform Resource Locator

For a link to work, it needs an address. URLs are that address system — a standardized format for locating anything on the internet.

http://info.cern.ch/hypertext/WWW/TheProject.html — the first URL ever written.

Three parts: the scheme (http://) says what protocol to use; the host (info.cern.ch) gets resolved by DNS to an IP address; the path locates the specific resource on that server. Every URL you've ever typed follows the same structure.

Invention 3

HTTP — HyperText Transfer Protocol

The protocol that governs how browsers and servers talk to each other. Simple, stateless, request-response.

The first version — HTTP/0.9 — was one line:
GET /page.html
The server sent the HTML back. No headers, no status codes, nothing else. Done.

Status codes came with HTTP/1.0: 200 OK means success. 404 Not Found means the page doesn't exist. 500 Internal Server Error means the server broke. Every developer eventually memorizes these.

Christmas Day, 1990

On December 25, 1990, Berners-Lee ran the first successful HTTP communication between a browser and a server over the internet. A browser on one machine requested a page. A server on another machine sent it back. It worked.

The web was born on Christmas Day, in an empty physics lab, with almost nobody watching.

Eight months later, on August 6, 1991, Berners-Lee published the first website: a page on info.cern.ch describing the World Wide Web project itself. He posted a link in the alt.hypertext newsgroup. Almost nobody noticed. He taped a note to the server machine: "This machine is a server. DO NOT POWER IT DOWN!!"

Still live

The very first web server — info.cern.ch — still responds today. The original page has been preserved and is accessible on the live internet. You can visit the first website in history right now. It loads in a fraction of a second. In 1991, loading it felt like magic.

What HTML Actually Is

HTML is not a programming language. You can't use it to make decisions, run loops, or respond to user input. It's a markup language — it describes structure and meaning, nothing more. 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World</h1>
    <p>This is a paragraph.</p>
    <a href="http://info.cern.ch">The first website</a>
  </body>
</html>

The browser reads this and decides how to display it. In 1991, "decides" meant whatever each browser thought looked right — no stylesheets existed. Every browser rendered pages differently. That visual chaos would take a decade to resolve.

Key Dates
Mar 12, 1989

Berners-Lee submits his proposal. His boss writes "Vague but exciting." on the cover and returns it. Work begins.

Dec 25, 1990

First successful HTTP communication. A browser requests a page. A server responds. The web works for the first time. It's Christmas Day.

Aug 6, 1991

First website goes live at info.cern.ch. Berners-Lee posts a link in a newsgroup. Almost nobody notices.

Apr 30, 1993

CERN releases the web royalty-free. The same month, the University of Minnesota starts charging for Gopher. One of them will be forgotten within two years.

Jun 1993

500 web servers exist in the world. By October 1994, there will be 10,000. By 1996, over 100,000. The growth never stops.

"The web is more a social creation than a technical one. I designed it for a social effect — to help people work together — and not as a technical toy."
— Tim Berners-Lee
