Chapter 02 — 1989 to 1995

The First Web

In 1989, a physicist at CERN wrote a proposal called "Information Management: A Proposal." His boss wrote "Vague but exciting" on the cover and returned it. That physicist was Tim Berners-Lee. What he was proposing would eventually become the most transformative communication medium in human history.

The Problem at CERN

CERN employed thousands of scientists from different universities, using different computers, constantly changing projects and roles. Every time someone left, their knowledge left with them. Documents existed but were scattered across incompatible systems.

Tim Berners-Lee's insight: the internet already existed. Computers were already networked. What was missing was a system that let documents link to each other across different machines — so reading one document could lead you naturally to related documents, wherever they lived.

He called this hypertext — text that contains links to other text. The concept existed (Ted Nelson theorized it in the 1960s), but Berners-Lee was the first to implement it over a live network.

The Three Inventions

To make the web work, Berners-Lee needed to solve three problems. He invented three things:

Invention 1

HTML — HyperText Markup Language

A simple language for structuring documents. Tags describe what content is: a heading, a paragraph, a link.

The killer feature: the <a href="..."> tag. A hyperlink. A way to point from one document to another, anywhere on the network. That single mechanic — click a link, load a document — is what made the web a web.

Invention 2

URL — Uniform Resource Locator

A standardized address format for anything on the internet. http://info.cern.ch/hypertext/WWW/TheProject.html — the first URL ever.

Three parts: the scheme (http://) tells the browser what protocol to use; the host (info.cern.ch) is resolved by DNS to an IP address; the path locates the resource on that server.

Invention 3

HTTP — HyperText Transfer Protocol

The protocol defining how browsers and servers talk. Simple request-response. The first version, HTTP/0.9, was a single line:
GET /page.html
The server responded with the HTML. No headers. No status codes. Done.

Status codes came later: 200 OK, 404 Not Found, 500 Internal Server Error — every developer memorizes these.

The First Website

On August 6, 1991, Berners-Lee published the first website on the internet, running on a NeXT computer at CERN. He taped a note to the machine: "This machine is a server. DO NOT POWER IT DOWN!!"

The site described the World Wide Web project itself — what it was, how it worked, how to set up a web server. Meta from day one.

The first browser, also built by Berners-Lee, was called WorldWideWeb (later Nexus). It ran only on NeXT computers. It could display text and hyperlinks but not images. Images would have to wait for Mosaic in 1993.

Curious fact

The very first web server — info.cern.ch — still responds today. The page at info.cern.ch/hypertext/WWW/TheProject.html has been preserved and is accessible on the live internet. You can visit the first website in history right now. It loads in a fraction of a second. In 1991, loading it was an act of magic.

What HTML Actually Is

HTML is not a programming language. You can't write logic in it. It doesn't make decisions or repeat operations. It's a markup language — it describes structure and meaning. 

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World</h1>
    <p>This is a paragraph.</p>
    <a href="http://info.cern.ch">The first website</a>
  </body>
</html>

The browser reads this markup and decides how to display it. In 1991, that meant whatever the browser thought looked right — no stylesheets existed. Every browser rendered pages differently. That chaos persisted for years.

A Crucial Decision: Free to Everyone

On April 30, 1993, CERN released the World Wide Web technology into the public domain. No licenses. No fees. No restrictions. Anyone could build a web server or browser.

This single decision is why the web exploded and why it still exists as an open platform. If CERN had patented it — as they could have — the web might have become a niche academic tool owned by a European physics lab.

Curious fact

Gopher — a menu-based internet document system that predated the web — was actually more popular than the web in 1992. It was cleaner, faster, and easier to use. It lost because the University of Minnesota (which owned it) announced they would charge licensing fees in 1993. CERN made the web free at exactly the same moment. One licensing decision changed the history of human communication.

Key Dates
Mar 12, 1989

Tim Berners-Lee submits his proposal "Information Management: A Proposal" to CERN management. His boss writes "Vague but exciting." on the cover.

Dec 25, 1990

First successful HTTP communication between a browser and a server via the internet. The web works for the first time. It's Christmas Day.

Aug 6, 1991

First website goes live at info.cern.ch. Tim Berners-Lee posts it to the alt.hypertext newsgroup. Almost nobody notices.

Apr 30, 1993

CERN releases the web royalty-free. The most consequential open-source decision in history. The web belongs to everyone.

Jun 1993

500 web servers exist in the world. By October 1994, there will be 10,000. By 1996, over 100,000. The growth is exponential from day one.

"The web is more a social creation than a technical one. I designed it for a social effect — to help people work together — and not as a technical toy."
— Tim Berners-Lee
