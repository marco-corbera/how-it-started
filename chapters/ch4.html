Chapter 04 — 1995 to 2004

The Dynamic Web

Static HTML pages were a start. But the real web — where you log in, where the page knows who you are, where things happen without reloading — required the web to grow a memory and learn to run code. This is the era where the web stopped being a document system and became a platform.

The Stateless Problem

HTTP was designed to be stateless. Each request is independent. The server responds, closes the connection, and immediately forgets you exist. The next request comes in as if from a stranger.

This is elegant from a protocol standpoint — simple, reliable, scalable. But it creates a practical problem: how does a website know you're logged in? How does a shopping cart persist between pages?

Core Concept

Cookies: How the Web Remembers You

Invented by Lou Montulli at Netscape in 1994. When a server responds, it can include a small piece of data in the headers: a cookie. Your browser stores it. On every subsequent request to that server, the browser sends it back automatically.

The server says: "here's a token identifying your session" → browser stores it → browser sends it back every request → server looks it up: "oh, it's Marco." Stateless HTTP suddenly has memory. This is still how login works today.

Server-Side Scripting: Pages That Think

The first web pages were static: a file on disk, served as-is. The same HTML went to every visitor. To change it, you edited the file by hand.

Server-side scripting changed this. Instead of serving a static file, the server runs a program that generates the HTML dynamically — different content per user, data pulled from a database, responses to form input.

Early tools were crude: CGI scripts in Perl, spawning a new process per request. Then came PHP (1995) — designed specifically for embedding code inside HTML. PHP made dynamic pages accessible to a much wider audience. By 2000, it powered a massive portion of the web. It still does.

Curious fact

PHP was originally created by Rasmus Lerdorf in 1994 to track visits to his online resume. He called it "Personal Home Page Tools." It was never intended to become a programming language. It just grew into one because people kept adding features. PHP now powers about 77% of all websites with a known server-side language, including WordPress, which runs ~43% of the entire web.

Flash: The Web's Greatest Detour

Macromedia Flash let developers create rich animations, games, and interactive experiences that HTML simply couldn't do. For about a decade, Flash was everywhere: animated intros, browser games, music players, entire websites built with custom motion and sound.

Flash powered the first wave of web video. It powered early YouTube. It gave us Homestar Runner, Newgrounds, and an entire generation of internet culture. It was the closest thing the web had to a real multimedia runtime.

But Flash was proprietary (Macromedia, then Adobe), a security disaster, terrible for accessibility and SEO, and required a plugin you had to manually install and constantly update. In 2010, Steve Jobs published "Thoughts on Flash" and refused to allow it on the iPhone. HTML5 arrived with native video, canvas, and animation. Flash's death was inevitable from that moment. Adobe killed it December 31, 2020.

Curious fact

The Flash game "The Impossible Quiz" (2007) was played by an estimated 100 million people. Thousands of beloved Flash games and animations — 15+ years of internet culture — went dark on December 31, 2020 when Flash died. The Flashpoint project has archived over 70,000 Flash games and 8,000 animations to preserve them. An entire era of internet art rescued by volunteers.

The Forum Era Matures

The late 90s had phpBB. The early 2000s saw the golden age of internet forums. vBulletin (2000), Invision Power Board (2002), and dozens of others competed. Every niche community had its own domain and its own forum software.

Forum culture in this era became genuinely complex social systems:

  • NeoGAF — the gaming forum where industry professionals posted alongside fans. Breaking news from game developers. Leaked info. Scandals. Unbans and permabans as public theater.
  • Something Awful — launched 1999, paid registration to post. Quality control through friction. The forum that produced much of early internet culture including early memes, YouTube Poop, and many creators who'd go mainstream.
  • Slashdot — tech news aggregator with comments. Invented the concept of comment moderation systems. "Slashdotted" (when your server dies because Slashdot linked to you) entered the vocabulary.
  • 4chan — launched 2003, anonymous. No accounts, no history, ephemeral threads. Complete chaos. And somehow, incredibly influential.

The Dot-Com Boom and Bust

Between 1997 and 2000, investors poured hundreds of billions into internet companies. The logic: the web would change everything, first-movers win, profitability irrelevant.

Pets.com raised $82.5 million in an IPO and went bankrupt nine months later. Boo.com burned $188 million in 18 months and collapsed. The sock puppet became a mascot of speculative excess.

In March 2000, NASDAQ peaked at 5,048. Then it crashed. By October 2002, it had fallen to 1,114. Five trillion dollars in market value — gone. Hundreds of dot-coms folded. Thousands of engineers were suddenly unemployed.

What survived: Amazon, Google, eBay, PayPal — companies with actual products and users. The bust killed the hype. It didn't kill the web.

XMLHttpRequest: The Quiet Revolution

In 1999, Microsoft added XMLHttpRequest to Internet Explorer 5 — a small API that let JavaScript fetch data in the background without reloading the page. For years, almost nobody noticed. Then Google launched Gmail in 2004. Then Google Maps in 2005. Pages that felt like desktop apps. Jesse James Garrett named the technique Ajax in 2005. The word spread. The next era had arrived.

Key Dates
1994

Cookies invented by Lou Montulli at Netscape. The web gets a memory. Also: the first online pizza order (Pizza Hut). The commercial web is real.

Aug 9, 1995

Netscape IPO — a company with no profits valued at $2.9B. The dot-com era officially begins. The word "internet" enters mainstream consciousness.

Mar 10, 2000

NASDAQ peaks at 5,048. The dot-com bubble is at its maximum. Within two years, $5 trillion in value evaporates.

Jan 15, 2001

Wikipedia launches. The radical idea that strangers can collaboratively write a reliable encyclopedia. It becomes one of the greatest knowledge projects in human history.

Oct 1, 2003

4chan launches, anonymous image board. No accounts, no memory, total chaos. Becomes one of the most culturally influential sites in internet history.

Apr 1, 2004

Gmail launches with 1GB of storage (competitors offered 4MB). The Ajax interface feels like a desktop app. The web as a platform is no longer theoretical.

"The web was supposed to be about documents. But people kept trying to build applications in it. Eventually, the platform caught up with the ambition."
