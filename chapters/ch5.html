Chapter 05 — 2004 to 2010

Web 2.0

Someone had to name the shift. Tim O'Reilly did it in 2004 with "Web 2.0." It wasn't a new software version — it was a new idea of what the web was for. Not a medium you read. A platform you used. Not content you consumed. Content you made.

Ajax: The Technique That Changed Everything

Ajax — Asynchronous JavaScript and XML — was a pattern: use JavaScript to fetch data in the background, update just the part of the page that changed. No full reload. No waiting. Pages that felt alive.

Gmail (2004) felt nothing like a web page — it felt like Outlook. Google Maps (2005) let you drag and see new tiles flow in seamlessly. The demonstration effect was immediate. Every major web app adopted Ajax within two years. The user experience of the web changed fundamentally. 

// The humble XMLHttpRequest — still underneath every fetch() call
var xhr = new XMLHttpRequest();
xhr.open('GET', '/api/data', true);
xhr.onreadystatechange = function() {
  if (xhr.readyState === 4 && xhr.status === 200) {
    var data = JSON.parse(xhr.responseText);
    document.getElementById('result').textContent = data.message;
  }
};
xhr.send();

jQuery: Tame the Browser Chaos

John Resig released jQuery in 2006 with a clear promise: "write less, do more." It solved a real problem — browsers implemented JavaScript APIs wildly differently. What worked in Firefox failed in IE. What worked in Safari was subtly wrong in Opera.

jQuery abstracted all of it. One API that handled every browser. Cleaner syntax for DOM manipulation. Plugin ecosystem. At its peak, jQuery was used on over 75% of all websites. It's still on roughly 70% today.

Curious fact

jQuery's $ function became so ubiquitous that for years new developers assumed $ was part of JavaScript itself. When they learned it wasn't, it was genuinely surprising. John Resig wrote the first version over a single weekend and presented it at BarCampNYC in January 2006. He was 22.

The Social Web Is Born

Web 2.0 was also when the web became about connecting people, not just serving content. The timeline of the social era is worth memorizing — it happened fast:

  • 2003: MySpace. Customizable profiles, background music, "top 8" friends. Peak teen culture. 100 million users by 2006.
  • 2004: Facebook from a Harvard dorm. Real names. Clean. No glitter. Within a year it's spreading beyond campus.
  • 2005: YouTube. Upload a video, anyone can watch it. Acquired by Google in 2006 for $1.65 billion — at the time considered absurd.
  • 2006: Twitter. 140 characters. Nobody understood what it was for. Everyone became addicted.
  • 2007: Tumblr. Reblogging. A blogging platform that felt like a social network. Where fandoms lived.
  • 2008: Stack Overflow. Q&A for developers. Forums reimagined with voting and accepted answers. Becomes the backbone of software knowledge.

The shift was profound: ordinary people became publishers. The web was no longer written by a technical class and read by everyone else. Everyone could write now. Everyone did.

Curious fact

The first YouTube video — "Me at the zoo" — was uploaded on April 23, 2005, by co-founder Jawed Karim. It's 18 seconds long. He's standing in front of elephants. The video currently has over 300 million views. Comments are disabled because they "detracted from the viewing experience." Even the founders of YouTube couldn't handle YouTube comments.

JSON Quietly Wins

Ajax used XML. XML is verbose, hierarchical, tedious to parse in JavaScript. JSON (JavaScript Object Notation) was simpler: data as native JavaScript objects, readable without a parser. 

// XML (exhausting)
<user><name>Marco</name><role>developer</role></user>

// JSON (obvious)
{ "name": "Marco", "role": "developer" }

JSON won. By 2010, almost all web APIs used it. REST — a set of conventions for HTTP APIs using standard verbs (GET, POST, PUT, DELETE) — became the default way to expose data on the web.

Forums vs Social Media: The Transition

The Web 2.0 era is when forums started losing ground. Not immediately — forums peaked around 2005–2008. But Facebook Groups could reach the same people without requiring them to visit a separate domain. Reddit (2005) centralized discussion with voting. Twitter took real-time conversation. The feed model beat the forum model.

What forums offered: depth, history, threading, indexability. A great thread from 2007 is still findable and readable today. What social media offered: everyone in one place, no friction. People chose friction-free.

Key Dates
Feb 4, 2004

Facebook launches from Mark Zuckerberg's Harvard dorm. Initially "TheFacebook." Real names required. Clean design. Within a year, most Ivy League schools are on it.

Apr 23, 2005

First YouTube video uploaded. "Me at the zoo" by Jawed Karim. 18 seconds. The age of user-generated video begins.

Jun 23, 2005

Reddit launches. Links + voting + comments. Communities organized around interest, not geography. The "front page of the internet."

Oct 9, 2006

Google buys YouTube for $1.65 billion in stock. Considered wildly overpriced at the time. Currently YouTube generates ~$30B/year in revenue.

Mar 21, 2006

First tweet sent by Jack Dorsey: "just setting up my twttr." Twitter officially launches in July. Nobody knows what it's for. Everyone starts using it.

Jan 9, 2007

Steve Jobs announces the iPhone. "An iPod, a phone, and an internet communicator." The mobile web era begins. The desktop web starts its slow decline as the primary access point.

Sep 13, 2008

Google Chrome launches. Fast. Simple. V8 JavaScript engine. Within 3 years it becomes the world's most used browser. IE era ends.
